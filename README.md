# 如何安装

0. 下载本附加包和Nebula on the ship
1. 安装Nebula on the ship
2. 启动游戏（如果是安装模组后的第一次启动的话）
3. 然后打开游戏根目录
4. 打开Addons文件夹
5. 将下载到的zip放进去再打开游戏即可。
